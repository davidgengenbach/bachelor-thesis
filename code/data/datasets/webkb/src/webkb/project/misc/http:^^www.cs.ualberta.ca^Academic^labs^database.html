Research Activities

The group associated with the Laboratory for Database Systems Research are involved in a number of research activities including the development of distributed object-oriented database systems, deductive databases, distributed database management, the application of neural networks to database problems, and the development of databases to support software engineering activities. The ongoing projects are the following:
  • TIGUKAT Distributed Objectbase Management System
  • Multimedia data management
  • DISIMA Distributed Image Database Management
  • Deductive database systems
  • Diorama/DIOM Interoperable Information System
  • Adaptive logic networks
