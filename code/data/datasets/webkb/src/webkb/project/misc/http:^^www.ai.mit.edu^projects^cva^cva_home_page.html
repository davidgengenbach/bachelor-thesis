CVA Group Home Page

Welcome to the Concurrent VLSI Architecture (CVA) Group's World Wide Web Home Page. The MIT CVA group is affiliated with both the Laboratory for Computer Science, and Artificial Intelligence Laboratory. The CVA Group investigates methods for applying VLSI technology to information processing problems. Ongoing projects address issues in parallel computer architecture, parallel computer software, interconnection networks, special purpose processor design, and VLSI design.
