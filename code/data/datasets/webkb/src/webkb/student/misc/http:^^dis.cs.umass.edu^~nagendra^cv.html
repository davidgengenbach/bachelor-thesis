Systems Building Experience

Multi-agent Systems Worked with a multi-agent design system called TEAM for creating steam condenser designs. Extended the TEAM system to enable learning organizational knowledge. Also worked with a simulator called TAEMS for modeling and studying coordination issues in multi-agent system. Extended this system to enable it to learn coordination. In addition, conceived, designed and built a system for distributed case-based reasoning. (COMMON LISP/CLOS).

Networked Information Gathering Worked on a team building an agent-based networked information gathering system. Was resposible for the planner component and the integration of all the system components. (COMMON LISP/C)

Machine Learning Systems Built a number of machine learning systems in both LISP and C. Programmed ID3, IBL, and CBR systems, wrote code for Genetic Algorithms and designed and built a flexible neural networks simulator in C. (LISP/ C).

Pattern Recognition Systems Project leader for a team that built a handwritten numerals recognition system for automated bank check processing. Was actively involved in building various components in addition to managing the team.

Information Retrieval Built a complete IR system as a part of Information Retrieval course. The system creates inverted indices, retrieves and ranks documents and displays them to the user. (C).

Expert Systems Built an expert system for Psychiatric Evaluation using the VP Expert shell (200 rules).

Information Extraction and Natural Language Processing Built a small IE system for the ``terrorist events'' domain using methods for statistical NLP for information extraction.

Compilers & OS Implemented a complete two-pass compiler for a considerable subset of MODULA to generate assembly code for MIPS R3000 architecture. This includes developing a scanner, parser, semantic processing and assembly language code generation. Also wrote code for device drivers and a shell.
