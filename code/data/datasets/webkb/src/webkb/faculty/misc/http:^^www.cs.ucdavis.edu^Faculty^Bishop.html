Matthew A. Bishop

Assistant Professor

Professor Bishop’s main research area is computer security, in which he has been active since 1979. He is especially interested in analyzing and improving the security of existing systems (especially techniques for detecting and preventing security flaws in design and implementation), network security, formal models of access control, user authentication, and cryptography. He has also worked extensively on the security of the UNIX operating system. Professor Bishop is a member of the Privacy and Security Research Group, and chaired the first two UNIX Security Workshops.
