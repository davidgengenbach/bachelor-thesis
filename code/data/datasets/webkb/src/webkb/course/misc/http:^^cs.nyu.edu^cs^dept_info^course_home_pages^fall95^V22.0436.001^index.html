Comments on the final

The exam is today (thursday) 2pm -- 3:50pm.
One good problem would be like what we did the last day of class. The exam defines a new instruction (in class it was jmp), states the format of the instruction (for jmp it was 6-bit opcode 26-bit address), gives the semantics, and asks you to modify the datapath to include support for this function. You would probably need to add one or more new control signals (for jmp the control signal was called jump).

Mano type questions will be something like 1/4 -- 1/3 of the test.
