Course Description:
Designed for students who program proficiently in Pascal, BASIC, Fortran, or some other high-level programming language. Begins with an intensive introduction to the C programming language and then covers all the materials in CLA CS 112.

Work Load:
Heavy, fast. This is an intensive course that covers the topics of CS 111 and CS 112 in one semester.
