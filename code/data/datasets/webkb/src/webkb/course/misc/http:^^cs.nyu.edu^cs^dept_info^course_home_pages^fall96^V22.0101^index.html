Printing Postscript Files

The following are postscript files. Click a given link and then save the appropriate file to disk. Exit Netscape and then double click the Main icon in Windows. Select the "DOS in a Window" icon to get the DOS prompt. Change the drive to C: . If you saved your program as ftp.ps, print it by typing copy/b ftp.ps prn WARNING: If you try to print the ps file by first terminating Windows, you will not get the required output.
