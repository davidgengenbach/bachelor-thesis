  • Printing Lecture Notes that are in Postscript format

    For files that are in postscript format, you need a postscript previewer. You should be able to view them on the SparcStations. If you want to print them, while viewing them, select "save" and save as a file with a ".ps" extension, for example: lect1.ps

    You can then send the file to the printer by typing: lpr lect1.ps
