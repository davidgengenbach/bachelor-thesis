CS 237

Design and Implementation of Programming Languages

CS 237 is a one term course on compiler optimizations being taught at Caltech. This year the course is being taught by Mary Hall, a visiting professor from Stanford. The course will cover the basics of compiler construction, with emphasis on optimization techniques and parallelizing compilers.
