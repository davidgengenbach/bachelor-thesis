COS 333: Advanced Programming Techniques (Spring 1995)

COS333: Advanced Programming Techniques

Important Announcement

Course Description:

Techniques for programming in the large and small will be discussed. Theories which have been developed to aid in the writing of clear concise programs will be debated for the writing of programs in both domains. following the lecture the class will examine the text of a large program in order to understand its structure and learn from the writing style of its author(s). We will begin with relatively short programs and evolve into large systems ending with an exploration of the UNIX operating system.
