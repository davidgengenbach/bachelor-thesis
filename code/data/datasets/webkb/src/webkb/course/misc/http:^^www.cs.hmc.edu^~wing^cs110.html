Catalog Description

Principles of computer architecture,from logic design through processor and memory structure. Introduction to parallel computers and computer networks. Prequisite:Computer Science 60. 3 credit hours.

Required Textbooks

Computer Architecture: A Quantitative Approach,by J. Hennessy & D. Patterson.

Syllabus

  1. Organization Of A Basic Computer
    Machine Instructions
  2. Computer Architecture
    Data Representations
    Register Organization
    Instruction Formats and Addressing Modes
    Instruction Sets
    Flow Of Control
    Bus Structures
    Input/Output

  3. Memory Hierarchy
    Cache
    Main Memory
    Virtual Memory

  4. Computer Arithmetic

  5. Digital Logic Circuits Of Some Basic Components

  6. Microprogramming And RISC

  7. Machine Organization Concepts For Increased Performance

  8. Survey Of Some Advanced Computer Architecture
    Parallel Computers
    Data Flow Machine

Project

Besides regular homework and programming assignments, the students are required to do a case study and make a presentation.
