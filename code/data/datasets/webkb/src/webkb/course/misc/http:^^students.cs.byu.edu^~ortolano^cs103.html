Welcome to the CS 103 Homepage



Computer Science 103 or Elementary Computer Applications was developed by Dr. Larry Christensen to help students from all majors to aquire basic computer skills. CS 103 is taught by computers at a level that all students can understand due to C.A.I (Computer Aided Instructions) developed also here at BYU. CS 103 is a fun class with the best TA's in the Computer Science Dept. so the students will have all the attention needed during the course.

These are some basic skills that the student will have by the end of the course:

    • Knowledge of Computer organization
    • Knowledge of Information Systems
    • Use of computers for problem solving
    • Programming skills
    • Internet Applications
    • TOPICS FOR TEST#2
