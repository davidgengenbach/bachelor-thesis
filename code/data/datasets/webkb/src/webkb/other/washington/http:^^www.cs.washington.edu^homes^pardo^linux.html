Linux Journal

Linux Journal is the monthly magazine for and about the Linux community. It is published by SSC, Inc., makers of those cute Pocket References for shells, C and Unix libraries, vi, emacs and so on. Linux Journal articles range from novice questions and answers to kernel programming.
