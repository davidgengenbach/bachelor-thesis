Finding clusters quickly in parallel

Brendan Mumey

Department of Computer Science
University of Washington, Seattle, WA 98195, USA
brendan@cs.washington.edu

January 27, 1995

Abstract:

This report describes the implementation of a new parallel algorithm for finding clusters point sets of Euclidean space. A new solution to a special case of the bichromatic closest pairs problem is used by the algorithm. The implementation was done in an environment for distributed tree computations on the Kendall Square Research KSR-2 multiprocessor.
