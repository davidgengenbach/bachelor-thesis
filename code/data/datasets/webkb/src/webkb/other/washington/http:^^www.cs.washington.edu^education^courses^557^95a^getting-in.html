CSE 557: Intro to Computer Graphics

Autumn Quarter 1995

Getting into the class

If you are a CSE graduate student you should be able to sign up for the course directly.

Otherwise, if you have satisfied the prerequisites but are not currently a CSE graduate student, you may still try coming to the first class. We'll accommodate as many as we can, but we are severely limited by the number of machines in our instructional lab.
