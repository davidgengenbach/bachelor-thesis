  • Singh, Stone, and Thiebaut, "A model of workloads and its use in miss-rate prediction for fully associative caches", IEEE TOC, 41:7, 811-25, July 1992.
  • Martonosi, Gupta, and Anderson, "MemSpy: analyzing memory system bottlenecks in programs", Performance Evaluation Review, 20:1, spec. issue, pp. 1-12, June 1992.

  • A Comparison of Trace-Sampling Techniques for Multi-Megabyte Caches, R. E. Kessler, Mark D. Hill, David A. Wood, IEEE Transactions on Computers, June 1994.
