Real-Time Systems: Simulators
University of Massachusetts, Amherst


Spring Real-Time Scheduling

The Spring real-time scheduling simulator supports the study of various real-time scheduling algorithms. This simulator integrates with the flexible manufacturing research being done in the CARTS lab. This is a multi-level simulator that deals with resources at two levels of abstraction (at the high level: products to manufacture, raw materials, etc. and at the low level: computational and data resources) and their linkage.


Real-Time Active Database Experimental Simulator

Real-Time Active Database Experimental Simulator (RADEx) is a real-time, active, temporal database simulator.


Current Work

Enhancing both the Spring scheduling simulator and the RADEx simulator top the list of current work in this area. Other interests include making these simulators accessible via NII (the information superhighway).
