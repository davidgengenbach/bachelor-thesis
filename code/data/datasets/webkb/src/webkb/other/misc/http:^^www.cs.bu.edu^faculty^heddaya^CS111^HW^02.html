As usual, homeworks are due at the beginning of class.

  1. The following program writes a large letter I on the screen: 
    	#include <stdio.h>

	#define HEIGHT 17

	main()
	{
		int i = 0;
	
		printf("\n\nIIIIIII\n");
		while (i < HEIGHT)
		{
			printf("  III\n");
			i = i + 1;
		}
		printf("IIIIIII\n\n\n");
	}
    1. Type, compile, and run this program
    2. Describe in a sentence or two of your own words what the program does.
    3. Write a similar program that prints a large letter C on the screen.

  2. (Extra-credit/Honors)
    Write a program that prints an X (the letter) that is 25 lines high, using as few printf statements as you can.
