The University of Massachusetts Amherst

You've reached the World Wide Web Server at the University of Massachusetts Amherst, oldest and largest campus of the University of Massachusetts system. This server provides a general overview of the University campus, has links to other campus Web servers, and holds campus information updated regularly.

Are you new to the World Wide Web? We have some new users instructions and a more thorough introduction to the Web .
