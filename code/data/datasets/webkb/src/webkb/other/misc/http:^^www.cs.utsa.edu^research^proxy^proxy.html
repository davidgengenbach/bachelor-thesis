The Proxy Sharing Proxy Server Project

The Proxy Sharing Proxy Server Project is an investigation into the application of wide-area demand-driven replication techniques to sharing WWW resources. The main difference between us and the competition is that our caching scheme is non-hierarchial. So far we have a design paper, presented in May at the ETACOM 96 conference in Portland, Oregon. Our implementation, based on the CERN proxy server, is coming along.
