How to Create Your Own Home Page

If you are a graduate student in the JHU Department of Computer Science and are interested in creating your own home page please see our on-line instructions. If you are in the Part-Time Computer Science program, please contact Homewood Academic Computing for support of your home page.

The document NCSA's Beginner's Guide to HTML provides basic information on HTML which is the file format you will use to create your own home page. The following are some references to how to write ''good'' HTML:
