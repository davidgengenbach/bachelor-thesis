New! Multi-Threaded JRM simulator

Downloading this will create a JRM.out.hqx. You might need to decode this by double clicking on the .hqx file. The executable is JRM.out. This is a runnable zip file and not a droplet. You launch it by double clicking on the .out file. You must first reboot (the simulator,not your mac :-) and then load the file to be executed by using the LOAD button on the simulator.
