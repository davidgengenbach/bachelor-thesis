Announcements

Note: The times of the demo sessions have been changed, because B78 Upson will be closing early on Monday (8pm).

Demo Sessions introducing the Dylan Interpreter, CoNote and Netscape will be held in the B7 Upson CIT Mac Lab, Monday and Tuesday September 2 and 3 at 6:30, 7:00, and 7:30pm. Each session is about 1/2 hour.

Problem Set 1 has been handed out (on the CoNote server). You need a CoNote user id and password to get the problem set, and to submit your solution.
If your Web browser has cached parts of an old NOODLLE, you may see strange behavior. The best thing to do at the beginning of a session is to start a new Web browser and to clear its disk cache.
There are a number of bugs in the Java implementation that comes with Netscape 3.0 for PowerPC Macintosh. The old Macintosh (also called 68K) version does not have this same problem. The solution is if you're working on a PowerMac to install the 68K version. The Netscape installer is in the CS212 course-ware folder on the Upson B7 Macs. Run the installer, and install the 68K version and use that instead of the one on the machine by default. We're in touch with Netscape and have let them know about this problem.
The due date for Problem Set 3 has been moved to Saturday night (19 Oct) at 2AM. (This is really early Sunday morning.)
