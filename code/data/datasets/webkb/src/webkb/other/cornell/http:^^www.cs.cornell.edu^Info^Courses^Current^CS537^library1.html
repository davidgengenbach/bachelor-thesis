Material in Engineering Library

The Engineering Library is located at the NW corner of the engineering quadrangle. It contains the following books on reference, and the following journals in the stacks. Also, you can check the on-line catalogs there, or via the WWW.

Reference Books

  • Korth & Silberschatz: Database System Concepts. McGraw-Hill, Second Edition, 1991.

  • Michael Stonebraker: Readings in Database Systems. Morgan Kaufmann, Second Edition, 1994.

  • Elmasri & Navathe: Fundamentals of Database Systems. Benjamin-Cummings, Second Edition, 1994.

  • Gray & Reuter: Transaction Processing: Concepts and Techniques. Morgan Kaufmann, 1993.

Conference Proceedings

  • ACM SIGMOD Conference on the Management of Data

  • The International Conference on Very Large Databases (VLDB)

  • The IEEE Conference on Data Engineering

  • ACM SIGACT-SIGMOD-SIGART Symposium on Principles of Database Systems

Journals

  • ACM Transactions on Database Systems

  • VLDB Journal

  • IEEE Transactions on Knowledge and Data Engineering

  • SIGMOD Record
