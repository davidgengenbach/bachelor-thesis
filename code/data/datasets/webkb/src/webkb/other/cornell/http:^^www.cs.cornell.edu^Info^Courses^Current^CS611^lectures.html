CS611: Semantics of Programming Languages

These notes are not intended for distribution outside of class.

Notes on Scribing:

  • Each student is responsible for scribing at least two class sessions. If your name isn't on here twice, send email to either Úlfar or me to get it on here twice.
  • I prefer that you generate a postscript (.ps) file, though html is also okay.
  • Scribe notes should be emailed to either Úlfar or me within one week of the class.
  • It's probably best to only produce one scribe document for each class, so coordinate with the other scribe.
  • I often have informal notes which you can work from. Check with me before you start to write up your notes.
