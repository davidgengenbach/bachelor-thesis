Using Projects


  • The first thing to do is to make sure that you have a directory "u:\programs\prog6\". You can do this by going to file manager. Make sure you can see a window displaying the u: drive. Also make sure that the "u:\programs" is highlighted in the left pane. Then go to "File->create directory", and make a directory called "prog6."
  • From now on, we will be using a project to manage our many programs. If you want to work on program 6, you will have to make sure that you've opened the project file for program 6. To create this project, we follow these steps:
    1. Go to "Project->new project". This should bring up a dialog box.
    2. In this dialog box, for "Project Path and Name," we will fill in "u:\programs\prog6\prog6.ide."
    3. In this dialog box, for "Target Type," make sure that "EasyWin [exe]" is selected.
    4. Then click on the button "Advanced". This should bring up another dialog box.
      1. In this dialog box, make sure that the "initial node" is ".cpp", and that .rc and .def are not checked.
      2. click okay to close this dialog. This should return us to the other dialog.
      3. Click okay to close this dialog.
  • We should now be in Borland, and there should be a new window with the title "project" that has something like this in it: 
      [ ] prog6.exe [.exe] .....
   |
    - prog6.cpp [.cpp]
    This tells us that the project wants to make the program "prog6.exe", and it knows that "prog6.cpp" is used to make that program. We want to add one more file to this project.
    1. First, click on "prog6.exe" to make sure it is highlighted.
    2. Next, press the "insert" key on the keyboard. This should bring up a dialog.
    3. Go to "u:\programs\prog6\" and enter "Appointment.cpp", and click okay.
    4. Notice that we did not add the "Appointment.h" file to the list. You should not add header (.h) files to this list.
    That should be it. You run your program in the same way, and debug it in the same way. To actually write the program, you will need to edit the files "prog6.cpp", "Appointment.cpp", and "Appointment.h".
