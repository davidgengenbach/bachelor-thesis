SHORE Project Beta Release (0.9.3)

Release Information

The latest release of the Shore software is version 0.9.3, the second beta release. It includes improved documentation, more complete implementations of many SDL features, many bug fixes, and ports to Solaris, HP-UX, Linux.

Online Documentation

The documentation that accompanies the release is available online. This documentation is still incomplete in some areas. We are actively working on improving it and welcome your suggestions.

Ftp Sources, Binaries, and Documentation

Compressed tar files containing the documentation, sources and binaries are available from: ftp.cs.wisc.edu in the directory shore/beta/. See the README there more details on which files to ftp.

Mailing Lists

Mailing lists exist for contacting users and the the developers of Shore. Please send questions, comments, and problem reports to
shore_support@cs.wisc.edu.

Shore Users Newsletter

The Shore Users Newsletter, is mailed periodically to the shore_all@cs.wisc.edu mailing list. The aim of this newsletter is to provide users with timely information on the SHORE project, especially the latest release. Much of the information will be excerpts of correspondence with users (ie. bug reports and help requests). We welcome suggestions for improving it. The newsletter is "published" every couple weeks.
