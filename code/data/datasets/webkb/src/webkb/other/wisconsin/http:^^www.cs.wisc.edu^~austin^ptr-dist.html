Pointer-Intensive Benchmark Suite

What is the Pointer-Intensive Benchmark Suite?

The Pointer-Intensive Benchmark Suite is a collection of pointer-intensive benchmarks that may be useful to people doing intensive pointer research.

How do I get the Pointer-Intensive Benchmark Suite?

The source release for the Pointer-Intensive Benchmark Suite, version 1.1, is available via anonymous ftp at "ftp.cs.wisc.edu" in the file "/sohi/Code/ptrdist-1.1.tar.Z". (Clicking on the highlighted file name will fetch the file to your machine.) Version 1.1 adds a unified Makefile scheme with built-in support for automated testing.

Who made the Pointer-Intensive Benchmark Suite?

I (Todd Austin) assembled the suite. Anagram was written Raymond Chen at Princeton, Backprop is from the SPEC '92 benchmark suite (source not included in this distribution) and it was written by Dean Pomerleau, BC is distributed by the Free Software Foundation, FT was written by Alain Kägi, KS was written by Todd Austin, YACR-2 was written by Scott Breach.
