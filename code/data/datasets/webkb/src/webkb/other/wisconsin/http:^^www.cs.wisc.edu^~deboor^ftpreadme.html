all icons on this page are clickable

These are the files that can be obtained by anonymous from ftp.cs.wisc.edu/Approx. The files are postscript, and are also available as compress(ed) files, as indicated by the subscript .Z, to be uncompress(ed) before using.
If you have trouble because of file contamination, specify binary as your
first command in ftp.
The files are in order of increasing age.
