Hints for Netscape

Netscape is another World Wide Web browser like Mosaic. There are a few differences between Netscape and Mosaic, but very few should have any impact on using the Web for this course.

Netscape is made by Netscape Communications Corporation. For more information about their software, check out their on line information.

You may find the Bookmarks feature of Netscape to be very useful to keep track of your instructor's page, assignments, or anything else you might discover.
