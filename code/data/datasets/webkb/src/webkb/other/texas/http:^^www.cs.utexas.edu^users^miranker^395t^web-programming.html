Homework 1,2,3 Web Programming:

You must become familiar (i.e. not an expert) with web programming. This includes, but is not limited to, HTML, CGI PERL and JAVA To ensure this you have the following set of assignments.
  1. Create a home page for yourself,. Due 9/12/96
  2. Develop a form using CGI embedding in PERL or something else. Due 9/17/96
    Best you should invent something. Otherwise, implement a four function calculator.
  3. Do the project in the "The One Hour Java Applet" Handout. Due 9/24/96
