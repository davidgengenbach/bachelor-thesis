They do not present a completed work, and should be viewed as such, however they do present new results and are worth exploring for anyone with an interest in UNITY.

The following four chapters are available:

  • Safety
  • Progress
  • Composition
  • Closure
The file SafetyProgress contains an unpublished journal article combining the safety and progress chapters, but omitting the exercises. This might be a good place to start reading about the revised UNITY theory.
