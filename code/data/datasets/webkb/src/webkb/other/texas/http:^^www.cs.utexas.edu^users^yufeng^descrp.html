Course Description

The MAIN goals of this course are
<1>to research CURRENT issues and their protocols in the dynamic environment of the Internet, and
<2> to study various implementation strategies in building network clients and servers
The SECONDARY goals are to develop your skills in:
<1> working in teams,
<2> writing a research paper,
<3> giving a short oral presentation,
<4> working on a large project, including some existing code,
<5> participating in class, and
<6> learning to effectively use the Web as an educational resource ( as oppposed to entertainment ).
