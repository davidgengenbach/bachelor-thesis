CS 381K: Programming Assignments

Programming assignments are to be done in Lisp; see Using Lisp on Local Machines. Notes on Lisp Style and Efficiency are available.

Lisp is easier to use through the Emacs editor.

Program files are provided, in the directory ~cs381k, to be used with the assignments; these are described by Program File Descriptions.

All programs must be individual work.

Assignments:

  • Lisp Timings and Performance
  • Symbolic Differentiation
  • Search: Solving the 8-Puzzle
  • Theorem Provers
  • Natural Language Access to Database
