P++

is an extension of C++ that supports GenVoca programming concepts.

  • A Programming Language for ... Generators Ph.D. Thesis, 1996.
  • P++: A Language for Large-Scale Reusable Software... WISR 1993.

  • P++: A Language for Software System Generators. TR-93-16.

Rosetta

is a project to build extensible and customized compilers for the domain of relational data languages.

  • Rosetta: A Generator of Data Language Compilers SSR 97.

  • ...Generation of Extensible Data Languages. Ph.D. Thesis, 1994.

Prairie

is an effort to build customized rule based query optimizers from components.

  • Synthesizing Rule Sets..., UTCS TR-96-05.
  • Prairie: A Rule Specification Framework... Data Engineering 1995.

  • Making Database Optimizers More Extensible Ph.D. Thesis, 1995.

Design Patterns

Many structural (Gamma) patterns can be recognized as end-products of large scale program transformations. Our work on design patterns explores the potential of automating structural patterns for software maintenance.

  • Program Transformations for Evolving Software ... OOPSLA'95.

  • Automated Software Evolution via Design Patterns... ISACC'95.

ADAGE

is an ARPA-sponsored DSSA project to build a (GenVoca) generator for avionics software.

  • Creating Reference Architectures... SSR '95.
  • The ADAGE Avionics Reference Architecture. CIA '95.

  • Domain Modeling... Symposium ECBS 1995.
